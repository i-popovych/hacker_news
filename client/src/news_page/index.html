Track Users Without Fingerprintinge
I want to be able to give users a 3 day trial period to use my product (a React + Node.js web app) without a sign-up. Seems that the best option online is fingerprint.com and I've learned the hard way that the free npm version is only 60% accurate. My other option is to save a uuid on the browser's local storage to track the user. This wont work if the user erased their history. I'd like to pause and ask the community - Is there a fundamentally better way to do an x-day free trial?
